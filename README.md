# Student Management System

## Overview
A JavaFX-based Student Management System developed to help instructors manage student records and grades through a simple graphical user interface.

## Features
- Login
- Dashboard
- Add Students
- Add Grades
- View Students
- Generate Reports

## Technologies Used
- Java
- JavaFX
